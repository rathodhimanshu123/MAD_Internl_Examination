# Smart Resume Builder & Job Application Tracker

## Project Overview

Smart Resume Builder & Job Application Tracker is a Flutter-based mobile application developed to help students and job seekers efficiently manage resumes and track job applications in one centralized platform.

The application provides offline-first support, structured data management, job application tracking, dashboard analytics, and resume-application mapping functionality.

---

# Problem Statement

Students and job seekers often face difficulties in:

* Managing multiple resume versions for different job roles
* Tracking applications submitted to different companies
* Monitoring application status updates
* Organizing job-related information in one place
* Accessing data without internet connectivity

This application solves these problems by providing a structured and user-friendly mobile platform.

---

# Objectives

* Create and manage multiple resumes
* Track job applications efficiently
* Monitor application progress using status tracking
* Provide dashboard analytics and insights
* Support offline-first functionality using local storage
* Improve organization during the job application process
